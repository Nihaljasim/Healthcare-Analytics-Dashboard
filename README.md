# Healthcare-Analytics-Dashboard
End-to-end Healthcare analytics project using MySQL, Excel, Power BI, and Tableau

## Project Overview

-An end-to-end healthcare analytics project built using MySQL, Excel, Power BI, and Tableau to analyze patient trends, doctor workload, clinical risk, and treatment costs.

-The project involves data modeling and KPI creation using SQL, followed by interactive dashboards with slicers and drill-downs to support resource planning, cost optimization, and decision-making.

-Key insights include workload imbalance across departments, high-risk patient segments, and cost variations by age group and diagnosis.
